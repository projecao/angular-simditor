# angular-simditor
Simditor

## Install

`npm install --save angular-simditor`

## Usage

```js

import Simditor from 'simditor'
import * as angular from 'angular'
import 'angular-simditor'

angular.module('app', ['angular-simditor'])

```

```html

    <div ng-app="app">
        <ng-simditor></ng-simditor>
    </div>

```
